
# git命令

* `git clone` - 克隆远程仓库
* `git status` - 查看当前状态
* `git add` - 添加文件，使其可以被git跟踪
* `git commit` - 提交版本
* 差异文件
    * `git diff` - 查看差异
    * `git diff 文件 >> xx.di` - 导出差异
    * `git apply xx.diff` - 使用差异文件

* 分支
    * `git branch` - 查看分支
    * `git branch -c 名字` - 创建新分支
    * `git switch -c 名字` - 切换分支
    * `git merge 名字` - 合并分支
    * `git branch -d 名字` - 删除分支

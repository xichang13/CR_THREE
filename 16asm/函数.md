# 函数

函数执行流程：

1. 参数入栈
2. 返回地址入栈，跳转到函数
3. 保存栈帧
4. 申请局部变量空间
5. 保存寄存器环境
6. 执行函数功能
7. 恢复寄存器环境
8. 恢复栈帧
9. 弹出返回地址，返回[平栈]
10. [平栈]

函数调用相关指令：

| 指令 | 说明 | 功能 |
| --- | --- | --- |
| call (near ptr) 标号 | 段内直接调用 | push 返回地址 <br> jmp 标号 |
| call REG <br> call near ptr / word ptr [EA] | 段内间接调用 | push 返回地址 <br> jmp 函数地址 |
| call far ptr 标号 <br> call dword ptr [EA] | 段间调用 | push |
| ret (n) | 段内返回 | pop ip <br> add sp, n |
| retf (n) | 段间返回 | pop ip <br> pop cs <br> add sp, n |

# masm 函数语法

```asm
函数名 proc [距离] [调用约定] [uses reg1 reg2...] [参数:word, 参数:word...]
    local 变量:word
    local 变量:word

    ; 函数体
    ret
函数名 endp
```

示例：

```asm
; 函数求和
sum proc near stdcall uses bx dx si di  a:word, b:word
    local sum:word
    mov sum, 0
    mov ax, a
    add sum, ax
    mov ax, b
    add sum, ax
    mov ax, sum
    ret
sum endp
```

* 距离

| 距离关键字 | 说明 |
| --- | --- |
| near | 段内调用 <br> 使用ret返回 <br> 调用时ip入栈 |
| far | 段内段间调用 <br> 使用retf返回 <br> 调用时cs ip入栈 |

* 调用约定

| 调用约定关键字 | 说明 |
| --- | --- |
| c | 调用方平栈 |
| stdcall | 被调用方平栈 |

* 局部变量

| 类型 | 局部变量类型 | 备注 |
| --- | --- | --- |
| db | byte | 可以直接赋值使用|
| dw | word | 可以直接赋值使用 |
| dd | dword | 不可以直接赋值使用 |
| dq | qword | 不可以直接赋值使用 |
| dt | tbyte | 不可以直接赋值使用 |

* invoke伪指令
  * 会生成参数入栈代码
  * 如果是c调用约定，会生成平栈代码
  * 如果是局部变量取地址，需要使用addr伪指令
  * 使用addr的时候，注意ax的使用

| 伪指令 | 说明 |
| --- | --- |
| offset | 取段内偏移 |
| addr | 取局部变量地址，使用LEA指令 |

```asm
invoke 函数名, 参数1, 参数2, ...
```

* 函数声明

```asm
函数名 proto 距离 调用约定 参数列表
```

示例：

```asm
sum proto near stdcall a:word, b:word
```

# 示例

```asm
assume ds:data

; 数据段
data segment
    g_str db "hello world", 0dh, 0ah, '$'
data ends

; 代码段
code segment

; strlen
strlen:
    ; 保存栈帧
    push bp
    mov bp, sp

    ; 保存寄存器环境
    push bx
    push si
    push di
    
    ; 参数出栈 (字符串地址)
    mov si, [bp+4]

    ; 初始化计数
    xor bx, bx

    ; 循环遍历字符串查找'$'
    lop:
        ; 将字符和 '$' 比较
        mov cl, [si + bx]
        inc bx
        cmp cl, '$'
        ; 不相等继续循环
        jnz lop

    mov ax, bx ; 将计数值存入 ax

    ; 恢复寄存器环境
    pop di
    pop si
    pop bx

    ; 恢复栈帧
    pop bp

    ; 返回
    ret 2

; 入口
entry:
    ; 设置数据段
    mov ax, data
    mov ds, ax

    ; strlen
    ; 参数入栈 (字符串地址)
    lea ax, g_str
    push ax
    call strlen ; 将下一条指令的地址入栈

    jmp exit

; 退出
exit:
    mov ax, 4c00h
    int 21h

code ends

    end entry
```

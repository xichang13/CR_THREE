资源

在汇编中使用资源，在vs里面把资源编译成.res文件，将vs里面的资源头文件变为汇编头文件。然后使用INCLUDE指令包含进来。在链接时，把.res文件和汇编文件一起链接。

库

通过.obj文件可以在c和汇编之间共享函数。或者使用动态库的方式使用对方的代码。

在vs中生成动态库格式如下：

extern "C" int __stdcall MySub(int n1, int n2) {
    return n1 - n2;
}

在汇编中使用动态库，使用includelib包含库文件，函数声明。

includelib Dll1.lib
MySub proto :dword, :dword

在汇编中生成动态库格式如下：

.code
MyAdd proc n1:dword, n2:dword
    mov eax, n1
    add eax, n2
    ret
MyAdd endp

DllMain proc hModule:HMODULE, ul_reason_for_call:DWORD, lpReserved:LPVOID
    mov eax, TRUE
    ret
DllMain endp
end DllMain

ml /c /coff Asm_DLL.asm
# 生成DLL
link /subsystem:windows /DLL /DEF:Asm_DLL.def Asm_DLL.obj

内联汇编

https://learn.microsoft.com/zh-cn/cpp/assembler/inline/inline-assembler-overview?view=msvc-170

在c中使用内联汇编，格式如下：

// 可以将c中的变量当成汇编中的变量使用
__asm {
    mov eax, 1
    mov ebx, 2
    add eax, ebx
}

在正常函数中有出栈入栈，裸函数中需要自己写，裸函数中没有任何的附加代码，所有的代码都需要自己写，格式如下：

__declspec(naked) void MySub() {
    __asm {
        push ebp
        mov ebp, esp
        mov eax, 1
        mov ebx, 2
        add eax, ebx
        mov esp, ebp
        pop ebp
        ret
    }
}
